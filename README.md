# NATS JWT .NET (Developers preview)

This is a developers preview of the NATS JWT library for .NET.
This library is a work in progress and is not yet ready for production use.

## About

A [JWT](https://jwt.io/) implementation that uses [nkeys](https://github.com/nats-io/nkeys.net) to digitally sign
JWT tokens for the [NATS](https://nats.io/) ecosystem.

## Development

You need to clone https://github.com/nats-io/nkeys.net at the same level as this repository
to build and develop on your machine until the library is published to NuGet.

See also https://github.com/nats-io/jwt
